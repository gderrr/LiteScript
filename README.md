# LiteScript

A simple procedural toy scripting language implemented in C++ designed to run simple scripts,
expanded with different functionalities to make the interpreter useful, with a unique programming philosophy.

:::warning ⚠️ **DISCLAIMER**
I’m not an experienced C++ developer, what started off as a silly side project grew into an
actually useful language with a lot of functionalities, so evidently excuse me for breaking
best practices and the low code quality.

Regardless, this language is mainly designed to run simple scripts regarding some topics that
are not trivial to implement in other languages like HTTP and database connections. It was
**NOT** designed to run real server side applications, so if you intend to do it use it at your own risk!
:::

## Documentation

Visit the official [LiteScript reference](https://gderrr.github.io/LiteScript/) page to read the documentation of all the features the language provides, as well as more information such as execution and the unique philosophy of the language.

## Installation

In the **dist** folder you can find the different versions of LiteScript available to download, with a guide on how to integrate the interpreter and its different dependencies.

## Examples

In the **examples** folder you can find different examples to showcase all the different functionalities of the language, note that some of them look odd because they were used by an older interpreter, which is pretty different from modern versions.
